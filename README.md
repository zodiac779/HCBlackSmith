# HCBlackSmith

## 🐋 How to use Docker run a HCBlackSmith 
1. Make an image
```sh
make build-image
```
2. Run HCBlackSmith
```sh
make start
```
## 🏃 How to run a HCBlackSmith
### x64 architect
1. Install library
```sh
pip install -r requirements.txt
```
2. Run HCBlackSmith
```sh
python .\BlackSmithHouseV1.pyc .\wallet.json
```
